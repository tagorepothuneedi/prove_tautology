To execute Tautology in RS method you need to run tautology_check_using_rs.py file.
You can execute as many formulas as you want.
Once you are done with execution press Ctrl+c to exit out of the program.

When it prompts provide a Propositional Formula, use variables from set of a,b,c,d,p,q,r,s
